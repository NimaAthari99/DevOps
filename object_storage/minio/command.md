## Basic Commands

| Command                           | What it does                              | When to use it        |
|-----------------------------------|-------------------------------------------|-----------------------|
| `./mc ls ADDED_MINIO`             | List buckets                              | **Often**             |
| `./mc mb ADDED_MINIO/NEW_BUCKET`  | Add NEW_BUCKET                            | **Often**             |
| `./mc rb ADDED_MINIO/NEW_BUCKET`  | Remove NEW_BUCKET                         | **Often**             |
| `./mc alias list`                 | List aliases                              | **Often**             |
