***[🔝 Table Of Contents](#table-of-contents)***

---

#### `hellovm1.yml` File
It's for  hellovm1 variables.

| Variable      | Value         |
|---------------|---------------|
| `sub`         | `hellovm1`    |
| `domain`      | `nima`        |
| `tld`         | `local`       |

As it's defined and shown in table, each defined variable has a value. Till now, just they are defined for fqdn of hosts. **`sub`** is sub domain and for each host is it's hostname. **`domain`** is our project's domain name. **`tld`** is third level domain name of our project.
